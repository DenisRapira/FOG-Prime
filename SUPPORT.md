# Support

Use GitHub Issues for reproducible application bugs and focused feature requests.

Before opening an issue:

1. Install the latest release from this repository.
2. Restart Windows and run FOG Prime again.
3. Confirm WebView2 Runtime and Base Filtering Engine are available.
4. Remove secrets and personal network information from screenshots and logs.

Security vulnerabilities must be reported privately according to `SECURITY.md`.

The project cannot guarantee compatibility with every provider, VPN, antivirus, corporate firewall, anti-cheat, or modified Windows installation.
