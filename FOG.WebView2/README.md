# FOG Prime

Автоматическая WebView2-оболочка FOG Prime. Пользователь видит только три общих этапа: система, соединение и доступ.

## Запуск

Готовый файл:

```text
FOG.WebView2\release\FOG Prime.exe
```

## Prime-сценарий

При открытии приложение автоматически:

- проверяет обязательные файлы, WebView2 Runtime, PowerShell, curl и списки;
- получает одно системное UAC-подтверждение при открытии;
- скрыто перебирает до четырех стратегий без ручного выбора;
- после каждой попытки параллельно проверяет Discord и YouTube;
- останавливается на первом рабочем режиме и показывает короткий итог;
- не показывает пользователю названия `.bat`, пути, логи или инженерные настройки.

## Служебный режим проверки

Чтобы открыть приложение без автозапуска стратегии:

```powershell
$env:FOG_PRIME_NO_AUTOSTART='1'
.\FOG.WebView2\release\FOG Prime.exe
```
